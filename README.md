# Clone du projet avec submodules

### À la main
```
    git clone https://github.com/voblivion/ENSEEIHT.git
    cd ENSEEIHT

    git submodule init
    git submodule update
```

### Automatiquement
```
    git clone --recursive https://github.com/voblivion/ENSEEIHT.git
    cd ENSEEIHT
```
